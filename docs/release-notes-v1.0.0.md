# v1.0.0

## 中文

### 亮点

- 极轻量、零依赖：纯系统原生脚本 + 文本配置
- 自动同步流程：定时执行 `add → commit → pull(rebase) → push`
- 多仓库、多分支同步：首次自动生成 `repos.txt` 与 `branches.txt`
- 跨平台脚本：Windows / macOS / Linux
- 双日志机制：完整日志 + 最近轮次轻量日志

### 适用场景

- 个人笔记、文档与代码仓库自动备份
- 单人多设备长期同步
- 需要定时自动保存工作进度

### 使用提醒

- 不适合多人同时编辑同一文件的协作场景
- 建议先在小仓库试运行，再逐步扩大同步范围

## English

### Highlights

- Ultra-lightweight and zero dependencies: native system scripts + txt config
- Automated sync cycle: scheduled `add → commit → pull(rebase) → push`
- Multi-repo and multi-branch sync with auto-generated `repos.txt` and `branches.txt`
- Cross-platform scripts: Windows / macOS / Linux
- Dual-log design: full history log + lightweight recent-cycle log

### Best For

- Auto-backup of personal notes, docs, and code repos
- Long-term sync across multiple personal devices
- Periodic automatic progress saving

### Notes

- Not suitable for multi-user concurrent editing on the same files
- Recommended to start with small repos before scaling up
